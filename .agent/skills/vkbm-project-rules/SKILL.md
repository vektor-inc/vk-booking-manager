# vkbm-project-rules

このスキルは、このリポジトリに固有の運用方針とコーディング規約をまとめたものです。
公式スキルの上に重ねて適用し、プロジェクト固有のルールを優先します。

## いつ使うか
- このリポジトリで作業を開始するとき
- PHP/JS/CSS/ブロック/REST/テストの変更があるとき

## 手順
1. 参照資料を読み、共通方針と設計思想を理解する
2. コーディング規約とセキュリティ指針を適用する
3. デザイン実装ルールとテストルールに従う
4. 公式スキルと競合する場合は、このスキルの方針を優先する

## 概要
WordPress 上で動作する予約管理プラグインです。  
美容・整体・レッスンなど、複雑なサービス形態や複数のスタッフ運用に対応し、予約の取りやすさと現場の回しやすさを両立しています。

## 想定ユーザー

1. サイト制作・運営をする本職のウェブ制作者がサロンなどのウェブサイト制作時にサイト内に設置
2. WordPressもある程度知識がある中小企業診断士やコンサルタントが集客・コストカットで提案
3. ITに強いサロンオーナー・スタッフ

## 予約機能の特長

### 複雑な営業・シフト形態に対応！、予約可能な枠のみ表示

サロンなどにおいて、店舗によって以下のように状況はさまざまです。

- 毎週の休業日に加えて第2・第4火曜日などの定休日
- 祝日・休日の前日の異なる営業時間
- 通常スケジュール以外の臨時休業および臨時営業時間変更
- 同日でも午前・午後の営業など複数の営業時間
- スタッフ毎に異なる出勤日・勤務時間（Pro版のみ）
- 提供サービス毎に異なるサービス提供時間

このような __複雑な営業・シフト形態でも予約可能な枠のみユーザーに表示__ されます。

### スタッフ指名予約（Pro版のみ）

ユーザーが任意のスタッフを指名し、スタッフ毎に指名料を指定する事ができます。  
※ 指名料が発生しないサービスメニューの設定も可能です。

### スタッフ自動アサイン（Pro版のみ）

また、指名なしの場合は対応可能なスタッフが自動的にアサインされます。

### 現場の運用を考慮した設計

ユーザーが店舗に電話で予約してきた場合も店舗側でスムーズに予約登録する事が可能になっています。

### 店舗側が内容を確認してから確定する仮予約モード

ユーザーが予約した時点では一旦仮予約として登録され、店舗側で内容を確認して、必要に応じて時間やスタッフのアサイン、金額などを調整した上で予約確定に指定するモードを用意しています。

### 予約及びキャンセル期限設定

予約やキャンセルを受け付ける期限を設定できます。また、メニュー毎にも異なる時間を指定する事が可能です。

### 顧客のカルテとしても使用可能

* 各予約情報は施術後にメモや施術写真を登録可能です。
* ユーザー毎の過去の予約一覧を表示する事も可能です。

## WordPressプラグインとしての特徴

### 簡単セットアップ

店舗毎の情報はそれぞれ異なるためいろいろ必要ですが、このプラグイン用に複数固定ページを作成したりする必要はなく、予約ブロックを固定ページなどに一つ配置するだけで完結します。

必要な情報が登録されていない場合はダッシュボードにアラートが表示されますので、それに従って登録してください。

### 単体の予約システムとしても使用可能

WordPress の固定ページなどに、このプラグインの予約ブロックを一つ配置するだけで動作するため、WordPress で作られたウェブサイト内での運用はもちろん、予約ブロックだけをトップページに配置した WordPress で、単体の予約システムとして運用する事も可能です。

#### 単体使用を想定した権限も用意

予約システム単体で使用した場合にサロンオーナーが混乱しないように、予約関連以外のメニューは表示されない権限を用意しています。

---

## できない事

### 外部サービスとの予約データの連携

#### 他サービスとの自動連携

現段階では他サービスとの自動連携機能はありません。他の予約サービスで予約がはいった場合はこちらに手動で登録する運用を想定しています。

## 予約フロー（概要）
1. メニュー/スタッフを選択
2. 日付・時間を選択
3. 予約内容を確認
4. ログイン/新規登録
5. 予約確定

## 管理機能
- 予約一覧で日時/メニュー/担当/ステータスを確認
- カレンダー表示で当日・月間の埋まり具合を把握
- 予約詳細の編集・ステータス管理に対応
- 指名なし予約の割り当ても後から可能

## スタッフ・メニュー管理
- スタッフの公開/非公開、指名可否、指名料を設定
- 担当可能メニューを紐付け、適切な候補だけ表示
- 表示順の調整に対応
- メニューの料金/所要時間/予約締切/バッファを設定
- タグ/グループで整理、公開/非公開やアーカイブ対応

## 導入メリット
- 電話予約の手間を削減
- スタッフの空きが見える化され、機会損失を低減
- 予約データを一元管理し、運営を効率化

## 対象業態
- 美容室 / サロン / 整体
- パーソナルレッスン / 教室
- スタッフ指名が発生するサービス全般
